# Bundled template assets

- `templates/apis_standard_19_v2.json` is the bundled Apis right-forewing
  19-landmark template.
- `reference_guides/apis_standard_19_v2_landmark_guide.png` is a visual-only
  landmark placement guide for that template.

The guide image is for manual placement reference only. It does not contribute
saved coordinates, is not stored with submitted specimens, and is not a
repository reference specimen.
