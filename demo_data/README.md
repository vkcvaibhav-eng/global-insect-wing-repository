# Demonstration data

- `templates/apis_standard_19_v2.json` is the bundled Apis right-forewing
  19-landmark template.
- `reference_guides/apis_standard_19_v2_landmark_guide.png` is a visual-only
  landmark placement guide for that template.
- `sample_right_forewing.png` is a deterministic synthetic schematic generated
  by `python scripts/generate_demo_image.py`.

The synthetic image is not a taxonomic reference specimen. The guide image is
for manual placement reference only and does not contribute saved coordinates.
These are non-sensitive test data for exercising upload, coordinate, review,
repository, and export behavior.
