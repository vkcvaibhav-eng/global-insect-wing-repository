# Demonstration data

- `templates/apis_v1.json` is an illustrative, versioned teaching template.
- `sample_right_forewing.png` is a deterministic synthetic schematic generated
  by `python scripts/generate_demo_image.py`.

Neither asset is a validated anatomical or taxonomic reference. They are
non-sensitive test data for exercising upload, coordinate, review, repository,
and export behavior.
